Functions in JavaScript

Functions are reusable blocks of code. They can take parameters and return a value.

Function declaration

javascript
function sum(a, b) {
  return a + b;
}
sum(2, 3); // 5

Arrow function

javascript
const sum = (a, b) => a + b;
const double = x => x * 2;
