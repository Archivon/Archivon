Variables in JavaScript

Variables store values for later use. In modern JavaScript we use let and const.

const and let

const cannot be reassigned; let can.

javascript
const name = "Maria";
let age = 20;
age = 21;   // ok
// name = "John";  // error!

Common types

Number, string, boolean, array and object:

javascript
const num = 42;
const text = "hello";
const active = true;
const list = [1, 2, 3];
const obj = { key: "value" };
